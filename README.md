\# API Integration Internship Project



\## Overview

This project demonstrates API integration using Node.js and Axios.  

It fetches data from two endpoints of the public \[JSONPlaceholder API](https://jsonplaceholder.typicode.com/) and displays it in a structured and filtered way.



\## Features

\- Fetches \*\*users\*\* and \*\*posts\*\* from the API

\- List all users

\- Filter users by city

\- Display a single user by ID

\- List first 5 posts

\- Display a single post by ID

\- Error handling for network issues, timeouts, and API errors
